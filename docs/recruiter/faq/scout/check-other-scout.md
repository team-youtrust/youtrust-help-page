# 他リクルーターのスカウト内容を確認したい

「オーナー」の権限が付与されているリクルーターに限り、確認ができます。

1. リクルーター管理画面の「スカウト」＞「チームのスカウト」
    ![管理画面のスカウトサイドバーメニュー](images/web-admin-scout-sidebar-menu.png){ width="80%" style="display: block; margin: 0 auto;" }
2. リクルーター管理画面の「検索」または「気になる」または「候補者」＞各ユーザー詳細ページ＞「スカウト履歴」より、他のリクルーターが送ったスカウトをご確認ください。
    ![タレント詳細のスカウト履歴](images/web-talent-detail-scout-history.png){ width="80%" style="display: block; margin: 0 auto;" }

ただし、チャット（＝直接の友人へのメッセージや、募集へ「話を聞きたい」を押した人へのメッセージ）の履歴は閲覧できません。スカウトのみが閲覧できます。

[スカウトとメッセージの違いを知りたい](scout-vs-message.md)

### **権限での制限**

- 公式リクルーター「オーナー」以外の権限
    - 誰がいつスカウトしたかはわかるが、内容までは見えません。

### **スカウト閲覧できる条件**

- 自分が公式リクルーター権限を付与されたタイミング以前の他のリクルーターのスカウト
    - 原則、誰がいつスカウトしたか、スカウト内容ともに見えます。

!!! warning "例外"
    スカウト送信者が既に自社のリクルーターでなくなっている場合は、誰がいつスカウトしたかは見えるが、内容までは見えません。
