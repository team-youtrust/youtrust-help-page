# :material-target: メッセージを送れる範囲について知りたい

- 自分の「つながり」であるタレント
- 自分の「友達の友達（2次つながり）」であるタレント
- 同所属のリクルーターが「友達の友達（2次つながり）」であるタレント
- 自社のカンパニーページをフォローしているタレント
- 自分が出した募集に「話を聞きたい」を押したタレント
- 登録直後のホットなタレント

また、メッセージには「メッセージ」と「スカウト」の2つが存在します。

その違いについては下記ヘルプページを参考にしてください。

[スカウトとメッセージの違いを知りたい](scout-vs-message.md)

[スカウトが送れないと表示される](cannot-send.md)
