# :material-clipboard-check-outline: 企業プラン導入にあたっての事前準備

YOUTRUST企業プランをご契約頂きまして、誠にありがとうございます！！

ご利用開始前に、スムーズに準備を進めるべく、事前準備でご対応頂きたいことをおまとめしております。

---

## 1. 社内向け登録資料のご案内

まずは社内向けにYOUTRUSTを企業プランで利用することを伝え、アカウントの登録をしてもらいましょう！

以下の資料をコピーし自社向けに編集した上で、より多くの方に漏れなく登録頂けるよう呼びかけましょう。

[社内向けYOUTRUST登録案内 資料](https://docs.google.com/presentation/d/1yjW9kCWRQkdRcjS6SLHVm89tVsZ6_CcHPlhwq4eEzrw/edit#slide=id.g14816945ff1_0_147)

![友達とつながる案内資料](images/web-friend-connection-guide.png)

スカウトのデータベース拡大のために、より多くのメンバーが登録し、つながりを広げることが重要です。アカウントの登録だけでなく、プロフィールの入力やつながり申請も忘れずに依頼しましょう。

## 2. 公式リクルーター権限の設定

アカウントの登録が終わったら、公式リクルーター権限の設定をしましょう。

リクルーターに設定したい方が決まったら、営業、またはカスタマーサクセスの担当に権限追加を依頼しましょう。

[公式リクルーター権限の追加方法を知りたい](https://help.youtrust.jp/recruiter/how-to-add-recruiter-account)

権限追加は企業様の管理画面上で可能ですが、はじめの1名を管理メンバーとして運営より登録いたします。お名前もしくはYOUTRUSTマイページのURLをご共有ください。

## 3. ご利用開始前アンケートへの回答

キックオフMTGでより最適な運用のご提案のため、運用開始前のアンケートにご回答をお願いいたします。（回答時間：約5分）

[運用開始前アンケート](https://forms.gle/wfMLfYPcK9ddrTm16)

頂いた情報に関しては、社内共有のみに活用させていただき外部への共有はいたしません。

## 4. キックオフMTGの設定

カスタマーサクセス担当との、キックオフMTGの設定をお願いいたします。

営業担当からご案内いたしました連絡方法にてカスタマーサクセス担当よりご挨拶とキックオフのご案内を行います。

**キックオフMTGの内容(基本：1時間)**

- ご支援内容のご説明
- 採用計画や注力ポイントのヒアリング
- 運用体制のご相談
- 運用計画の仮決め
- 管理画面の操作方法のご案内
- 次回以降のMTG、ネクストアクションの整理

## 5. カンパニーページの作成

お打ち合わせまでに、貴社のご紹介用の**カンパニーページ[(完成イメージ)](https://youtrust.jp/companies/9537e5ddcac66d1a017fa5058e9c5c4f)**を作成させていただきたく以下の入力フォームに必要事項のご入力をお願いいたします。

[カンパニーページの情報入力フォーム](https://docs.google.com/forms/d/e/1FAIpQLSeUxYZZzea6VKi8mLm1g9jn_kkQTvtbw6KBVMMvQJvoBvbFwA/viewform)

※貴社のセキュリティの設定次第ではフォームが開けない場合がございますのでその際はお手数ですが以下の必須項目のデータ各連絡手段にてカスタマーサクセス担当まで送付頂けますと幸いです。

!!! info "必要項目"
    1. 企業名（正式名称）
    2. 企業情報（文字数制限なし）
    3. ホームページURL
    4. Slug（カンパニーページのURL[ https://youtrust.jp/companies/ ]の後ろに続く文字列をご指定のものに変更可能です。使用可能文字は小文字英数字とハイフンです。）
    5. 会社ロゴ（推奨比率は1:1）※形式はpng
    6. カバー画像（推奨サイズは16:7）※形式はpng
