# :material-pencil-outline: カンパニーページを編集したい

リクルーター管理画面の「カンパニー管理」より編集が可能でございます。

![管理画面サイドバーのカンパニー管理](images/web-admin-sidebar-company-management.png)

「カンパニー管理」内の「[カンパニーページ](https://admin.youtrust.jp/company_page)」より各種設定・変更いただけます

![カンパニーページ編集画面](images/web-admin-company-page-edit.png)

企業名(カンパニーページ名)の変更はできかねますので、会社名変更などやむを得ず企業名(カンパニーページ名)を変更したい場合は担当CXまでお問い合わせください。
