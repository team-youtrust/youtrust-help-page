# :material-new-box: 「新着ユーザー順」とは何か知りたい

並び替えにおける「新着ユーザー順」とは、**新しくスカウト対象となった順**のことを意味します。

リクルーター権限を新しい方に付けた時や、リクルーターやリクルーターの友達それぞれが誰かと繋がった時などに、新しくスカウト可能になるユーザーがいると思います。

それらの方は、「ログイン日時が近い順」や「登録日時が新しい順」などでの並び替えでは埋もれてしまう可能性があるため、**「新着ユーザー順」**としてまとめて確認することができるようにしております。

管理画面のトップページでも、直近1週間で新しくスカウト対象となった方の人数が表示されますので、そちらからもアクセスが可能です。
