# アプローチできる候補者を増やしたい

# 友達とつながる

2次のつながり（友達の友達）を増やし、アプローチできる範囲を広げられます。
アプローチ範囲を広げるためには、信頼している方に**「つながり申請」**を送って直接の友達を増やしましょう。

2次のつながり（友達の友達）の中では、候補者になるだけではなく、意欲が見え、スカウトを送ることもできます。

[つながりを増やしたい](../connection/increase.md)

# YOUTRUSTで友達とつながるために

### プロフィールを記入する

SNS連携や過去の経歴を入力をすると、友達候補が自動的に増えていき、友達が増えます。
また、あなたの友達・同僚がYOUTRUSTに新しく登録すると、同じく候補として表示されます。

プロフィールには、自分が何者であるのかを**具体的に**記入することが大事です。
また、現在の所属には**正式名称**で会社名を記入しましょう。

プロフィールを記入することは、公式リクルーターとしての信頼UPにもつながります。
※スカウトを受け取ったユーザーは、「スカウト送信者のプロフィール」を最初に確認します。
