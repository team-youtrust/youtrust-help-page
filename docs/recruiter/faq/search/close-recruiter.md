# :material-account-network: 「つながりの近いリクルーター」とは何か知りたい

「つながりの近いリクルーター」とは、同所属のリクルーターの中で、対象のユーザーと「友達」もしくは「友達の友達」のリクルーターのことを指します。

![つながりの近いメンバーの表示](images/web-close-recruiter-connection.png)

表示されるアイコンは、左から順に、対象のユーザーとの関係性が「友達」→「友達の友達」という並びで、その中でも共通の友達が多いリクルーターが優先される仕様となっております。

自分以外にも、対象のユーザーとつながりのあるリクルーターがいる場合、**より関係性が近いリクルーターからスカウトを送るよう依頼する** といったことが可能ですので、ぜひ効果的にご活用ください。
