# :material-content-save-outline: 検索条件を保存したい

## 検索条件の保存方法

1. キーワード検索から検索したい条件を設定

    絞り込み条件、フリーワードともに適応されます。

2. 「この検索条件を保存する」からラベル名をつけて保存

    保存条件はリクルーターごとの管理となっております。

    同じ所属の他のリクルーターには共有されませんのでご了承ください。

![管理画面の検索条件保存](images/web-admin-save-search-condition.png)

※複数の企業様にてリクルーターとなっている方は、所属先ごとの保存でございます。

A社で保存した条件はA社でのみ保存反映されます。
