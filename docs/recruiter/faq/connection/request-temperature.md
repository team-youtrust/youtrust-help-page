# :material-thermometer: つながり申請の温度感を知りたい

## つながり申請とアプローチ可能範囲の関係

YOUTRUSTは「2次のつながり」の範囲でリクルーティングが可能なサービスです。

つながりの数が多ければ多いほどプローチ可能範囲が広がりますので、つながり候補の方にはぜひつながり申請を送ってみて下さい！

## つながり申請を送りたくない場合

YOUTRUSTは「信頼できる関係」をもとにしたキャリアSNSなので、無理に全員につながり申請を送る必要はありません。

言い換えると、**信頼している方以外とは、無理につながる必要はありません。**

あくまでも実際に付き合いのある方や、同僚・信頼できる知り合いなどの範囲で申請を送る形で問題ありません。

## 注意点

友達ではないがスカウトを送りたいので申請をするといった、リクルーティングを目的とした知らない方へのつながり申請はNGとしております。

YOUTRUSTのサービス世界観、スカウトを受け取る候補者側の気持ちを考えてつながり申請を送る基準を決めていただくよう、お願いいたします。
