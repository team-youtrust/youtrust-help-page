# :material-fire: 登録直後のホットなタレントβ版とはなにか知りたい

ホームに、**YOUTRUSTに登録初期のおすすめタレントをランダムに5名表示しております。**

該当する**タレントにスカウトを送る場合は**、スカウト通数を消費しません。

![登録直後のホットなタレント一覧](images/web-hot-talent-list-with-bookmark.png)

### できること

「登録直後のホットなタレントβ版」に表示されるタレントに対しては、以下のアプローチが可能です。

- 意欲を含む**プロフィールの閲覧**
- **気になるリストへの追加**

    気になるリスト追加済みのタレントは、表示期間経過後も気になるリスト内に保持されます。

- **スカウトの送付**

    該当ユーザーとのスカウトルームに表示されている日時を過ぎると、通常通りスカウト通数を消費します。
    
    ![スカウト通数無料期間の表示](images/web-scout-room-free-period.png)

### こんな方におすすめ

- YOUTRUSTのアクティブなユーザーに出会いたい方
- 他の企業がまだアプローチしていない優秀な人に出会いたい方
- 既存のアプローチ範囲にはスカウト対象が少ない方
