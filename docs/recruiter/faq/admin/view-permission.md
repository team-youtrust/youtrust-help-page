# 管理画面を閲覧できる方

管理画面は**公式リクルーター権限が付与されている方のみ**が見ることが出来ます。

社内のどのメンバーが権限があるのかは、マイページ > マイメニュー > リクルーター管理画面 > 設定 > メンバー管理 より、確認いただけます。

[公式リクルーター権限の追加方法を知りたい](add-recruiter.md)

また、閲覧できる画面には、権限によって違いがございます。

詳細は下記ヘルプページをご確認ください。

[**リクルーター権限の各役割の違いについて知りたい**](role-differences.md)

# 管理画面を閲覧できるメンバーを変更したい

スタンダードプラス以上のプランでは権限の自由な変更が可能です。

それ以外のプランでは、退職など特別な場合を除き、公式リクルーター権限対象者を、契約期間中に社内の別メンバーに付け替えることはできません。

（契約プラン毎に設定された「公式リクルーター上限数」を超えていない場合には、契約期間中に上限まではメンバー追加は可能です。）

個別のご相談は担当CXまでお問い合わせください。
