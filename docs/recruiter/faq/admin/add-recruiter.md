# 公式リクルーター権限の追加方法を知りたい

1人目の公式リクルーターとなるユーザーに権限を追加したい場合は、弊社の担当CXもしくはSales宛に、公式リクルーター権限を付与したい方のマイページURLをご共有ください。

弊社にて設定後、画面上部の「管理画面」より[リクルーター管理画面](https://youtrust.jp/recruiter_team)にログインできます。

!!! warning "注意事項"
    - プロフィールの「現在の所属」に、**自社の「正式名称（※株式会社等を含む）」を追加**していないと、権限の追加はできません。
    - リクルーター権限の削除や付け替えは、スタンダードプラスプラン以上のプランであれば可能です。それ以外のプランでは、ご退職や異動等の特別な場合を除き原則不可となっています。

また、2人目以降の権限追加は、[管理画面の設定](https://youtrust.jp/recruiter_team/settings)から行なうことが可能です。

[設定](https://youtrust.jp/recruiter_team/settings) > 「+メンバーを追加」ボタンより、追加したいメンバーのマイページのリンクをコピーして、権限を設定の上追加してください。

!!! warning "注意事項"
    - 複数の企業にてリクルーター権限を利用したい場合や、現在ご利用中で一方を削除したい場合は、管理画面での対応はできません。担当のCXまでお問い合わせください。
    - 公式リクルーターになると、プロフィールに「公式リクルーター」のバッチが表示されるため、他のユーザーからは「所属している企業で採用活動をしている人」と捉えられます。プロフィール内の記載する内容はご注意ください。
        - 特に、目立ちたいがためにむやみに転職意欲を高くしていると、スカウト返信率が非常に下がってしまいますのでご注意ください。

権限の違いについては、下記ページをご確認ください。

[**リクルーター権限の各役割の違いについて知りたい**](role-differences.md)

その他、公式リクルーターになるとできることやプランによる差分については、下記ページをご確認ください。
