# :material-bell-outline: 管理画面内の通知について知りたい

管理画面の右上、ベルマークより通知を確認いただくことが可能です。

## 通知

通知に表示される内容とその対象は以下の通りです。

| **通知内容** | **対象者** |
| --- | --- |
| 評価の変更 | 担当者・過去の担当者・コメントしたことがあるメンバー・メンションされたことのあるメンバー |
| 進捗状況の変更 | 担当者・過去の担当者・コメントしたことがあるメンバー・メンションされたことのあるメンバー |
| 優先度の変更 | 担当者・過去の担当者・コメントしたことがあるメンバー・メンションされたことのあるメンバー |
| リストからアーカイブ | 担当者・過去の担当者・コメントしたことがあるメンバー・メンションされたことのあるメンバー |
| 社内メモの追加 | 担当者・過去の担当者・コメントしたことがあるメンバー・メンションされたメンバー・メンションされたことのあるメンバー |
| 期日の変更 | 担当者 |
| 担当者の変更 | 新しく割り当てされた担当者 |

※通知内容・通知対象者は随時アップデート予定

## お知らせ

お知らせに表示される内容とその対象は以下の通りです。

| **お知らせ内容** | **対象者** |
| --- | --- |
| 管理画面に関するYOUTRUSTからのお知らせ | 公式リクルーター |
| CSから公式リクルーター向けのお知らせ | 公式リクルーター |

※お知らせ内容・通知対象者は随時アップデート予定

## 意欲変更

意欲変更に表示される内容とその対象は以下の通りです。

| **意欲変更内容** | **通知者** |
| --- | --- |
| 気になるリスト・候補者リストに追加しているユーザーの転職意欲変更 | 担当者・過去の担当者・コメントしたことがあるメンバー・メンションされたメンバー・メンションされたことのあるメンバー・気になるリストに追加したメンバー・候補者リストに追加したメンバー |
| 気になるリスト・候補者リストに追加しているユーザーの副業意欲変更 | 担当者・過去の担当者・コメントしたことがあるメンバー・メンションされたメンバー・メンションされたことのあるメンバー・気になるリストに追加したメンバー・候補者リストに追加したメンバー |

※通知内容・通知対象者は随時アップデート予定
