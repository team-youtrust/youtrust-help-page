# 過去送付したスカウトがユーザー画面（メッセージ一覧）側で非表示になるケースを知りたい

過去送付したスカウトがユーザー画面（メッセージ一覧）側で非表示になるパターンは以下3パターンです。

1. 退職によりリクルーター権限を削除した場合
2. リクルーター権限の付け替えを行なった場合
3. YOUTRUSTをご解約した場合

※ユーザー画面（メッセージ一覧）側で非表示になった場合の実際のメッセージ画面について知りたい場合は下記ヘルプページをご確認ください

[**公式リクルーター権限を外した後の実際のメッセージ画面について知りたい**](../communication/after-permission-removed.md)
