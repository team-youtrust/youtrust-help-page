# :material-shield-off-outline: 公式リクルーター権限を外した後の実際のメッセージ画面について知りたい

公式リクルーター権限を外した後のメッセージ画面については、下記3つのケースで異なります。

- 退職などにより公式リクルーター権限を外した「元公式リクルーター」
- 元公式リクルーターからスカウトを受け取った「ユーザー」
- 公式リクルーター権限を引き続き付与されている「現公式リクルーター」

※送信スカウトが非表示になるケースを知りたい場合は下記ヘルプページをご確認ください

[**過去送付したスカウトがユーザー画面（メッセージ一覧）側で非表示になるケースを知りたい**](../communication/hidden-scout.md)

### 元公式リクルーターのユーザー画面

![元公式リクルーターのメッセージ画面（送信スカウト非表示）](images/web-message-sent-scout-empty.png)

- 公式リクルーターとして、スカウト活動をしていた方が公式リクルーター権限を外した際には、**元公式リクルーター**という扱いになり、ユーザー画面側でのスカウトが非表示になります。
- 元公式リクルーターはスカウトの閲覧・やり取りが不可になります。

### スカウト受信ユーザー側のユーザー画面

![スカウト受信ユーザー側のスカウトルーム](images/web-scout-room-after-permission-removed.png)

- 元公式リクルーターからスカウトを受信したユーザーは、文章の閲覧・返信は可能です。
- 元公式リクルーターが所属していた企業の契約終了後はスカウトを受信したユーザー側も返信不可となり、やり取りが終了します。

### 現公式リクルーターの管理画面

![現公式リクルーターの管理画面スカウト検索](images/web-admin-scout-search.png)

- 現公式リクルーターは、元公式リクルーターのスカウト文章を管理画面の「チームのスカウト」から閲覧可能です。
- ユーザーから返信があった場合は返信文章を閲覧可能ですが、お返事は不可となっております。その場合は新規でスカウトをお送りいただくようお願いしております
    - ※ユーザーから返信があった場合、現公式リクルーターが管理画面の「チームのスカウト」から返信できるよう今後アップデート予定でございます。
