# :material-coffee-outline: カジュアル面談とは何か知りたい

カジュアル面談とは、**選考を含まない面談**のことです。

企業が候補者を、候補者が企業を、お互いに理解し合うために、「企業の事業内容や採用背景」「候補者の方のキャリアの志向」などを話す場です。

あくまで**「面談」であり「面接」ではございませんので、カジュアル面談で合否を出すことはしません。**

また選考ではないので、「志望理由」などを伺うのは、候補者の方からすると違和感を覚えられるケースもございます。

より詳しい情報はCXチームにお問い合わせください。
