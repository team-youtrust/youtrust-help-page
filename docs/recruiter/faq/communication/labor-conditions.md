# いつ労働条件を示す必要があるのか知りたい

正社員、アルバイト、パート、派遣社員など「**雇用契約**」での募集の場合は、**選考前**の「労働条件の明示」が必須となっております。

必ずしも募集投稿の中に記載する必要はありませんが、「**正式な選考前**」に労働条件を明示することが必要です。必ず候補者と合意し、法律・ルールに則って選考を進めていくことが重要です。

また、副業や業務委託など「請負契約」「委任契約」「準委任契約」での募集の場合は、原則として上記の対象外となります。(実態が雇用契約である場合を除く)

どのような契約でも、誠実に・真摯に対応することが重要です。候補者の方と互いに合意を取り、選考や契約を進めましょう。
