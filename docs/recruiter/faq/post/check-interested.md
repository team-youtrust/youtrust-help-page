# 「話を聞きたい」を押してくれた人を確認したい

下記5つの方法でご確認いただけます。

1. サービスの通知欄
2. マイページの応募者管理
3. リクルーター管理画面の応募者管理ページ
4. リクルーター管理画面の候補者のリスト
5. Slack連携による通知

## サービスの通知欄

- YOUTRUSTの通知欄に、「〇〇さんが話を聞きたいを押しました」と通知されます。
- 通知から直接、お名前をクリックするとお相手のプロフィールページに遷移し、確認できます。

## マイページ応募者管理から確認

- マイページの募集タブから、自分が投稿した内容を確認できます。
- 募集投稿の場合、該当の投稿に遷移すると、「**話を聞きたいを押したユーザー**」を応募者管理から確認できます。
- サービス通知欄と同様に、その方のプロフィールページに直接飛ぶことができます。
- 応募者管理一覧から直接メッセージを開始できます。
    - 応募者のステータス管理が「対応済み」「対応不要」のいずれかを選択でき、応募者へは通知されません。

## リクルーター管理画面の応募者管理ページ

- 「話を聞きたい」を押したタレントは、リクルーター管理画面の「募集」内の「応募者管理」からご確認いただけます。
- 応募者名で検索や募集作成者で絞り込みが可能です。
- 応募日が新しい順に並んでいます。
- HERP連携していただくことで「話を聞きたい」したユーザー情報が自動で取り込まれます。

## リクルーター管理画面の候補者のリスト

- 「話を聞きたい」を押したタレントは、自動でリクルーター管理画面の候補者のリストに追加され、「話を聞きたい」をしたタレントのみに絞り込みが可能です。
- HERP連携していただくことで「話を聞きたい」したユーザー情報が自動で取り込まれます。

## Slack連携による通知

リクルーター管理画面「設定」＞外部サービス連携「[Slack連携設定（チーム設定）](https://youtrust.jp/recruiter_team/settings/team_slack_linkage)」より指定のSlackチャンネルへの通知設定が可能です。
