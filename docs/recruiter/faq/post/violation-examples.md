# :material-alert-circle-outline: 利用規約や法令に違反となる事例を知りたい

YOUTRUSTの利用規約、または職業安定法・男女雇用機会均等法などの法令に違反するサービス利用は、通常投稿・募集投稿(求人)・メッセージ問わず、また事前の通告なく運営チームが投稿を削除またはアカウントを停止することがあります。

特に**法令に反する求人をした場合、管轄当局から指摘・処罰対象となりうるのは、求人を掲載した主体=リクルーターの皆さん及び所属企業になってしまいます。**利用規約や法令に準拠した採用活動を行うようお願いします。

## 利用規約に反する事例

- 採用エージェントを業としている方が、自社以外の仕事・業務の紹介やあっせん、投稿をすること
- 他者との出会いや交際を主たる目的としたサービス利用
- ネットワークビジネス、マルチ商法、その他これらに類似する活動に関するもの

詳細は[利用規約](https://youtrust.jp/terms_of_service)をご覧ください。

## 法令に違反する事例

以下の内容を投稿・メッセージに含む場合は、法令違反になるケースがあります。最新の法令を確認し、準拠した投稿やサービスの利用を行ってください。

| 分類 | NG表現 | OK表現 |
| --- | --- | --- |
| 性別 | 男性歓迎／女性限定 | 原則、表記しない（※現状を表す事実表現は可。例：男性活躍中／女性社員多数在籍） |
| 性別 | 営業マン、主婦活躍中（※一方の性を想起させるため） | 営業スタッフ、主婦(夫)活躍中 |
| 年齢 | 25歳～35歳までの方、20代歓迎 | 原則、表記しない（※現状を表す事実表現は可。例：20代活躍中、20代がターゲットのブランドのマーケターを募集） |
| 人種・民族・国家・国籍・宗教 | 国籍不問／日本国籍の方歓迎／中国人歓迎 | 歓迎や不問表記であっても表記不可（※言語スキルに関して言及したい場合は、例：中国語に堪能な方、英語：ビジネスレベル以上） |
| 出身地・居住地 | 東京都中央区在住の方限定、1時間以内で通勤できる方 | 地元の方歓迎（※出身地や居住地の明記及び推察できる条件設定がNG。本人の実力と関係のない要素で、選考結果に優劣が出ることを防ぐためです。） |

また正社員・アルバイトなど**「雇用契約」を伴う募集**をする場合には、職業安定法に基づき**労働条件等の明示を適切に行う必要と責任が、求人主体の企業・組織・個人に課せられます**。
