# :material-link-variant: 投稿をカンパニーページに紐づけたい

投稿タイプや公式リクルーター権限の有無によって異なりますので、下記をご確認くださいませ。

### :material-clipboard-text-outline: カンパニーページに紐づく募集

カンパニーページのメンバーに登録された公式リクルーターからの「カジュアル面談」「メンバー募集」は、ご自身がカンパニーページのメンバーであれば自動でカンパニーページに紐づきます。

![募集作成時の企業選択画面](images/web-post-create-select-company.png)

### :material-pencil-outline: カンパニーページに紐づく投稿

カンパニーページのメンバーに登録された公式リクルーターからの、公開範囲を「全体公開+会社」に設定した投稿は紐づけが可能です。

!!! warning "2025年2月時点での注意事項"
    一度、カンパニーページへ紐付けた投稿は後から、編集で**公開範囲を変えることはできません**。

    公開範囲を「全体公開+会社」に設定した投稿は、投稿者の公式リクルーター権限を外した際や投稿者をカンパニーページメンバーから外した際にも、カンパニーページのアクティビティには**紐づいたまま**でございます。

![投稿の公開範囲「全体公開+会社」設定](images/web-post-public-scope-company.png)
