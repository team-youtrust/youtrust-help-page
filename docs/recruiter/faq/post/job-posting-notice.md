# :material-file-document-alert-outline: 雇用契約を伴う求人掲載時の注意点を知りたい

正社員・アルバイトなど「雇用契約」を伴う募集をする場合には、職業安定法に基づき労働条件等の明示を適切に行う必要と責任が、求人主体の企業・組織・個人に課せられます。

請負契約、準委任契約をはじめとした業務委託契約は職業安定法の対象外のため、労働条件等の明示は不要です。しかし契約と実態が異なる場合にはその限りではありませんので、**雇用実態に即した契約締結とルールの遵守をお願いいたします。**

## :material-information-outline: 労働条件とは？

雇用契約に関わる労働者の募集の際に、労働条件を書面の交付によって明示する必要があります。ただし求職者が希望する場合には、電子メール・YOUTRUST含む各種SNSチャット・FAX等によることも可能です。

最低限明示しなければいけない労働条件の項目は以下になりますが、詳細は厚生労働省作成の[リーフレット](https://www.mhlw.go.jp/content/001114110.pdf)や動画をご覧ください。

厚生労働省などからの最新情報を確認するようお願いします。

!!! warning "2024年4月1日、明示すべき労働条件が追加されてます！"
    求人広告のスペースが足りない等、やむを得ない場合には「詳細は面談時にお伝えします」などと付した上で、労働条件の一部を別途のタイミングで明示することも可能です。
    この場合、原則、面接などで求職者と最初に接触する時点までに、全ての労働条件を明示する必要があります。

!!! info "必ず明示する項目"
    業務期間 / 契約期間 / 試用期間 / 就業場所 / 就業時間 / 休憩時間 / 休日 / 時間外労働 / 賃金 / 加入保険 / 募集者の氏名又は名称 / 従事すべき業務の内容 / 始業及び終業の時刻 / 受動喫煙防止措置の状況 / 雇用形態(派遣労働者として雇用する場合)

    [募集採用担当者研修用動画「うっかり店長の求人広告」10分28秒](https://youtu.be/3mfflTXIO_E)

## :material-clock-outline: 労働条件等の明示タイミング

遅くとも正式な選考過程における初回面接（ 電話面接やビデオ面接を含む ）までの間に、労働条件等を明示するようお願いします。

原則は投稿内に労働条件等を含めていただき、スペース不足の都合などがある場合には「詳細は面接でお伝えします」などと記載の上、労働条件を明示するようお願いします。

詳細は厚生労働省作成の[こちらのリーフレット](https://www.mhlw.go.jp/content/001114110.pdf)をご覧ください。
