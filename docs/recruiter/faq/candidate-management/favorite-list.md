# :material-heart-outline: 気になるリストについて知りたい

気になるリストは、タレントプール機能です。

例えば、

- カジュアル面談をしたが、今は転職を検討しておらず、意欲の上がったタイミングで再度声をかけたい候補者Aさん
- 前職で一緒に働いており、優秀な方だと知っているので、いつかまた一緒に働きたいと思っている候補者Bさん

など、中長期的に採用し得る候補者の情報を管理することができます。

詳しい使い方は、下記ヘルプページや資料を参考にしてください。

[:material-clipboard-list-outline: 候補者を管理する](../guide/manage-candidates.md)
