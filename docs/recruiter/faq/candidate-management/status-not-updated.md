# :material-progress-alert: 候補者のリストの進捗状況が「スカウト済み」にならない

候補者リストの進捗状況が**自動で**「スカウト済み」に変わるのは、下記いずれかの場合のみです。

- スカウト送付時に、候補者リストに**追加していない**場合
- スカウト送付時に、候補者リストに追加済みで、かつ**進捗状況が「未設定」**の場合

そのため、スカウト送付時に、既に候補者リストに追加済みで、かつ進捗状況に「スカウト予定」等の何かしらのステータスを設定している場合は、**手動で「スカウト済み」に変更**していただく必要がございます。

**例：** 下記の図において、

- 「高見 朋恵さん」へスカウトを送付した場合、**自動で「スカウト済」に変わります**。
- 「西川 浩史さん」へスカウトを送付した場合、**自動で「スカウト済」に変わりません**。
