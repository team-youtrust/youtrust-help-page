# :material-bell-outline: リストに追加すると相手に通知されるのか知りたい

## :material-heart-outline: 気になるリストに追加した場合

プロフィールや候補者検索画面から「気になる」に追加した場合、**相手に通知が届きます。**

気になるリストでは、自分が相手に興味を持っているアピールなどにも活用することができます。

※気になるリストの名称は相手には通知されませんのでご安心ください。

![気になるリスト追加時の通知](images/web-list-add-notification.png)

その通知では企業名が表示され、カンパニーページへと遷移する動線が引かれております。そのため気になるリストに追加する前に、カンパニーページを作成しておくことを推奨いたします。

## :material-clipboard-list-outline: 候補者リストに追加した場合

プロフィールや候補者検索画面から「候補者」に追加した場合、**相手には通知されません**。
選考管理を進めるにあたって、安心して活用ください。

リクルーター管理画面内「候補者」のリストでのステータス変更なども同様に、相手からはわからない・通知は届かないようになっています。
