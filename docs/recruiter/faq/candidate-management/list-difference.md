# 「気になるリスト」と「候補者リスト」の違いを知りたい

## 気になるリスト

「気になるリスト」は、**タレントプール機能**です。

例えば、

- カジュアル面談をしたが、今は転職を検討しておらず、意欲の上がったタイミングで再度声をかけたい候補者Aさん
- 前職で一緒に働いており、優秀な方だと知っているので、いつかまた一緒に働きたいと思っている候補者Bさん

など、中長期的に採用し得る候補者の情報を管理することができます。

具体的には、下記のようなリストを作成することを推奨しております。

- 職種やポジションごとに気になる人
- 転職意欲「今は考えていない」人を含んだ、気になる人（＝意欲上がったらスカウトするためのリスト）
- 今後の組織体制を考えたときにいずれ声を掛けたい人

また、気になるリストに追加すると、追加した相手に**リストに追加された旨の通知が届きます**。

[リストに追加すると相手に通知されるのか知りたい](notification-on-add.md)

## 候補者リスト

「候補者リスト」は、**選考管理機能**です。

スカウト送付後、面談や面接等が進み、選考ラインに乗った方を管理することができます。

候補者リストに追加した際、追加した相手に通知は届きません。
