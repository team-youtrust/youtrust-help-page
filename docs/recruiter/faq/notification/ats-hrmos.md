# :material-connection: HRMOS採用との連携はこちら

## 連携の方法

HRMOS採用との連携をするためには、webhookトークンを発行する必要がございます。

webhookトークン発行については、お手数ですが下記申請フォームへのご入力をお願いいたします。

[RoboRoboリクルーティング 利用申請フォーム](https://docs.google.com/forms/d/e/1FAIpQLSdug4jgxkinpLC4p1AUEfKlOybIsxozFjQpUDprHOkVKQhLrg/viewform)

利用申請フォームご入力後、HRMOS採用よりトークン発行に関するご連絡が届きます。

### **YOUTRUSTのリクルーター管理画面での操作**

①**YOUTRUSTリクルーター管理画面＞[設定](https://youtrust.jp/recruiter_team/settings)＞外部サービス連携＞HRMOS連携**の「webhookトークン」欄に取得した媒体連携トークンを入力する
