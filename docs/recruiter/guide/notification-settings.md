# :material-bell-cog-outline: 通知設定を管理する

## :material-bell-check-outline: YOUTRUSTで活用すべき通知

YOUTRUSTサービス上の「通知」欄に届く通知の中でも、特にリクルーターの皆様に活用いただきたいものをご紹介します。

通知は「メール通知」「アプリ通知」「リクルーター管理画面の更新内容通知」と３種類あります。詳細はこちらのヘルプページをご覧ください。

[意欲変更通知について知りたい](../faq/notification/interest-change.md)

## :material-email-sync-outline: 通知をメール・Slackで受け取る設定

YOUTRUSTサービス内通知をメールまたはSlack、アプリではプッシュ通知で受け取ることができます。

Slack連携をすると通知を即時受け取りできるので、意欲の変わった瞬間の候補者へのスカウトや、返信のあったスカウトへの返信などにすぐに対応できます。

[Slackに通知を連携したい](../faq/notification/slack-integration.md)
