# 友達・つながりについて知りたい

YOUTRUSTでは、独自のつながり=サービス内のネットワークを築くことができます。

YOUTRUSTで直接つながっている人を「つながり」と呼びますが、同僚やビジネス上でお付き合いのある方も信頼できる人であれば、ぜひYOUTRUSTでつながってみてください。

「友達候補」は、各種SNSや勤務先情報、プロフィール情報の類似性を元に表示されますが、信頼している方以外とは友達になる必要はありません。

YOUTRUSTの友達が増えると、

1. **見える求人数が増える**
2. **もらえるメッセージが増える**

ようになり、キャリアのきっかけにつながる機会と巡り合いやすくなります。

ぜひたくさんの信頼できる人と、YOUTRUSTでつながってみてください！

## YOUTRUSTでつながって友達になる方法

### 1. 友達候補から探す

登録完了後は画面右上の「追加マーク」から友達候補を探すことができます。

SNS連携や過去の経歴を入力をすると、友達候補が自動的に増えていきます。また、あなたの友人・同僚がYOUTRUSTに新しく登録すると、同じく候補として表示されます。

つながりを申請すると、YOUTRUSTでのつながりを申請したことが相手に通知(メール+サービス内通知タブ)されます。

相手が申請を承諾して初めてYOUTRUSTでの「友達」になります。

プロフィールを充実させた後、定期的につながりリクエスト・友達候補を見に行ってみてくださいね！

[プロフィールの追加や編集をしたい](../profile/edit.md)

[友達候補に表示される条件を知りたい](suggestion-criteria.md)

### 2. つながりたい人のプロフィールページから直接「つながり申請」をする

コミュニティなどでつながりたい人を見かけたら、直接「つながり申請」をしましょう！（申請数の制限あり）

つながりたい人の名前をタップすると、その人のプロフィールページにつながります。右上「つながり申請」のボタンを押すと、つながり申請ができます。承認されるとYOUTRUSTの友達になります。

### 3. つながりたい人を検索して「つながり申請」をする

つながりたい人がいるが、Facebookのつながりがなく、友達候補にも見当たらないという場合は、検索してみましょう。

タイムラインの上にある検索窓に名前を入力すると、YOUTRUSTに登録しているユーザーから検索できます。

検索結果につながりになりたい人が表示されたら「つながり申請」をしましょう。承認されると友達になれます。
