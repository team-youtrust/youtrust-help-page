# 「ブロックする」について知りたい

特定のユーザーからのメッセージの受信、プロフィールの閲覧等を制限したい場合は、「ブロックする」を押すことで制限をかけることができます。

## 「ブロックする」とどうなる？

- つながり候補に表示されなくなります。
- ブロックしたアカウントは、あなたのプロフィールを閲覧できなくなります。
- ブロックしたアカウントは、あなたの投稿を閲覧・いいね・コメントできなくなります。また他のユーザーへのコメントも表示されません。
- ブロックしたアカウントは、あなたにメッセージを送ることができなくなります。
- ブロックしたアカウントとのチャットルームは消えます。
- ブロックしたことは相手には通知されません。
- アカウントをブロックすると、友達・フォローからも外されます。
- ブロックしたプロフィールを見たなどの閲覧通知情報が消えます。
- お気に入りに入っているユーザーからは削除されます。
- キャリアの一緒に働いていたユーザーに登録していたユーザーであれば、一緒に働いていたユーザーから削除されます。
- ユーザープロフィール画面の下にでる足跡情報も表示されなくなります。

※ ブロックするために相手のプロフィールを訪問する必要があります。しかし、ブロックした時点で、ブロックした相手方への「足跡の通知」は消える仕様となっています。

## 手順

ブロックしたい対象の人の[プロフィールページ](https://youtrust.jp/mypage?tab=friends)を開き、

- **アプリ**：右上の三点マーク
- **Web（友達）**：「友達」のボタン
- **Web（友達以外）**：左下の三点マーク

をクリックすると、「ブロックする」の項目が現れます。
