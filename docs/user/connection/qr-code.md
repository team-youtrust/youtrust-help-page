# :material-qrcode: QRコードでつながりたい

QRコードを使えば相手のプロフィールの表示やつながり申請を手軽に行えます。

新たに出会った人とQRコードをお互いに読み取れば、プロフィールを見ながらの自己紹介や、その他の連携されたSNSの交換も簡単に可能です。

## :material-cellphone: アプリ

ホーム＞右上の自分のアイコン＞右上のQRコードマーク

![アプリのQRコード画面](images/app-qr-code-screen.png)

下部の「QRコードをスキャンする」から相手のQRコードを読み取ることもできます。

## :material-monitor: Web

現在、Web版にはQRコード機能はありません。
