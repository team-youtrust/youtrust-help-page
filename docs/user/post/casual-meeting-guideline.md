# :material-coffee-outline: カジュアル面談Guideline

## :material-information-outline: 本ガイドラインについて

YOUTRUSTは信頼される人が最良のキャリアを歩めるように、つながりからキャリア・オポチュニティに出会えるSNSとして、ご利用いただいています。

転職を考えない日常期でも、趣味や働き方の温度感が合う人とカジュアル面談を通じて、これまで以上に気軽にキャリア形成のための"つながり"を作れるようになりました。

一方で、カジュアル面談の定義が曖昧なことで、多くの誤解や機会損失が生まれていることも事実でございます。

<span style="color: #00a5b5; font-weight: bold; font-size: 1.1em;">「もっとたくさんのキャリア・オポチュニティにつながるように」</span>

という思いを込めて、YOUTRSUTでのカジュアル面談のガイドラインを用意しました。

## :material-cancel: 禁止事項

ユーザーの皆様に安心してご利用いただけるプラットフォームであり続けるために、YOUTRUSTでは以下のような行為を禁止しております。

1. カジュアル面談に参加する方からの逆採用活動はご遠慮ください。
2. カジュアル面談で相手が求めていない一方的な営業行為はご遠慮ください。
3. 恋愛等の出会いを目的として面談を実施をすることはご遠慮ください。
4. 宗教活動または宗教団体への勧誘行為などはご遠慮ください。

詳しくは[サービス利用規約](https://youtrust.jp/terms_of_service)、[プライバシーポリシー](https://youtrust.jp/privacy_policy)をご確認ください。

禁止事項に当たる行為が報告された場合は、当社は募集コンテンツの非表示もしくは削除、またはアカウントの停止もしくは削除をすることがございます。
