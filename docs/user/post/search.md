# 検索について知りたい

YOUTRUSTのサービス内を検索することができます。

アプリとWebで検索できる対象が異なります。

- :material-cellphone: **アプリ**：ユーザー名
- :material-monitor: **Web**：ユーザー名、投稿内容、企業名
※企業名は[カンパニーページ](../community/company-page.md)のみヒットいたします。
