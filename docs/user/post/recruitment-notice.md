# 募集投稿時の注意点

YOUTRUSTでは、ユーザーの皆さんが自由に募集投稿を掲出し、その内容を見ることができます。

募集投稿をユーザーの皆さんが安心して見ていただき、素敵な出会いが生まれるよう、募集時には以下の点に注意して投稿してください。

## :material-scale-balance: 法令に則った募集

職業安定法や[募集・求人業務取扱要領](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/0000179543.html)などの法令を確認し、法令を遵守した募集を投稿・公開するようにしてください。

- 性別、年齢、人種、出身地、居住地、国籍等の区分に基づき募集対象を限定・除外・優遇する表現が含まれるものは、法令違反となります。
- 法令違反となる例
    - 20代限定（年齢制限）
    - 女性アシスタント（性別限定）
    - 日本国籍の方（国籍限定）
- 厚生労働省などからの最新情報を確認するようお願いします。

!!! warning "2024年4月1日、明示すべき労働条件が追加されてます！"
    求人広告のスペースが足りない等、やむを得ない場合には「詳細は面談時にお伝えします」などと付した上で、労働条件の一部を別途のタイミングで明示することも可能です。
    この場合、原則、面接などで求職者と最初に接触する時点までに、全ての労働条件を明示する必要があります。

## :material-file-document-outline: 労働条件等の明示

正社員・アルバイトなど「雇用契約」を伴う募集をする場合には、職業安定法に基づき労働条件等の明示を適切に行う必要と責任が、求人主体の企業・組織・個人に課せられます。

詳細は、以下のページをご覧ください。

[雇用契約を伴う求人掲載時の注意点](job-posting-notice.md)

## :material-close-circle-outline: 利用規約違反となる募集

YOUTRUSTの[利用規約](https://youtrust.jp/terms_of_service)を遵守し、募集投稿を投稿・公開するようにしてください。

- 利用規約違反となる例（一部抜粋）
    - 他社へのあっせんを行う目的での募集
    - 異性との出会い目的等、明らかにサービス範囲外のマッチングを目的とするもの
    - 反社会的勢力に関わるもの
    - 第三者の著作権、肖像権、名誉権、プライバシー権などの権利を侵害するもの
    - ネットワークビジネス、マルチ商法、その他これらに類似する活動に関するもの

## :material-alert-outline: 違反した募集を投稿していた場合

事前の通告なく運営チームが投稿を削除またはアカウントを停止・削除することがあります。
