# :material-plus-circle-outline: コミュニティの作成方法を知りたい

新規コミュニティ作成につきましては、当面の間は応募制になる予定です。
コミュニティ作成をご希望される方は、[こちら](https://forms.gle/CFF3FsHmodjsM9zD9)よりご登録ください。応募期間が決まりましたら追ってお知らせいたします。

定期的に新規コミュニティオーナーの募集を予定しておりますので、コミュニティ作成をご希望の方はYOUTRUST運営からのアナウンスをお待ちいただけますと幸いです。
