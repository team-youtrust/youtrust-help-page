# :material-exit-run: コミュニティの退出方法を知りたい

コミュニティメンバーは、参加コミュニティを任意のタイミングで退出することができます。

ただし退出後の投稿やコメントはコミュニティ内に残ります。

退出した後、再びコミュニティに参加したい場合は、コミュニティページから参加することができます。
