# :material-book-open-variant: コミュニティ ルールブック

!!! info "ルールブックとは？"
    すべてのユーザーさまが、YOUTRUSTコミュニティを楽しく・安心・安全に使うことができ、信頼できるつながりを深く・広くできるようになるための指針です。

最終更新：2025/07/14

みなさん一人ひとりで、健全で安心できるYOUTRUSTコミュニティを一緒に創って行きたいと思っています。

## YOUTRUSTとは

YOUTRUSTは自分のつながりの範囲でキャリアに関する発信をしたり、意欲を変更することで新しいキャリアのきっかけに出会える日本のキャリアSNSです。

コミュニティでは、職種や趣味、出身などの共通点をもとに新たなつながりを作っていただくことが可能です。

## YOUTRUSTコミュニティについて

YOUTRUSTのコミュニティは、興味関心のある話題の中でユーザー同士で交流し、情報を共有する場所になります。

コミュニティメンバーは、コミュニティ内の会話を通じて交流を深めることができます。コミュニティに参加したメンバーだけが、コミュニティの中で会話をすることができます。

YOUTRUSTコミュニティでは、**「安心して、安全に楽しむことができるコミュニティ」**を目指しています。

そのために以下のルールを守り、コミュニティをお楽しみください。

- コミュニティメンバー同士、お互いに敬意を持って接しましょう。
- コミュニティを通じて発生するイベントにおいて、個人情報やお互いが交流を深めるために開示したパーソナルな情報はコミュニティの中や個々人の間で留めるようにしましょう。
- その他、YOUTRUST全体の利用規約を守った上で、ご参加・コミュニケーションください。

[利用規約](https://youtrust.jp/terms_of_service)

## コミュニティメンバーとして

YOUTRUSTのコミュニティは、自分の興味関心とマッチすれば、どのコミュニティにも自由に参加・退出することができます。

- メンバー同士、お互いに敬意を持って気持ちの良いコミュニケーションを取りましょう。
- 自分の興味・関心があるものに参加するようにしましょう。退出も好きなタイミングで可能です。
- 言動が不適切なメンバーを見つけた時は、そのユーザーまたはコミュニティ投稿の「報告する」ボタンから、運営チームに報告をしてください。
- 信頼できるつながりを広げ、深めるためにも、参加したコミュニティでは積極的にリアクション（トピック作成・コメント）をしましょう。
- 情報商材・出会い系・相手の意思に反する営業活動・その他犯罪行為および利用規約に違反する行為は、YOUTRUSTサービス内に限らず、コミュニティを通じて生まれたイベント・オフ会等でも行わないでください。
- コミュニティでは「つながり」ではない方とも出会うことができる機会です。つながり申請は、コミュニティ内でお互いに信頼し合えるコミュニケーションを取り、関係が深まった後に送るようにしましょう。

## コミュニティオーナーとして

コミュニティオーナーの方は、以下内容を踏まえて、楽しく・安心・安全なコミュニティ運営をお願いいたします。

- あくまでご自身が興味・関心のあるトピックをコミュニティとして作成してください。「信頼」を損ねるような人脈づくり・ネットワーキング・集客だけが目的のコミュニティの作成は、ガイドラインに反します。
- トピックやコメントのメンバー間交流が盛んになるよう、積極的な運営・促進に取り組んでください。言動が不適切なメンバーを見つけた時は、そのトピック・コメントの3点メニューから削除することができます。
- 情報商材・出会い系・相手の意思に反する営業活動・その他犯罪行為を目的としたコミュニティの運営は、禁止しています。そのような懸念・疑いがある場合には、事前の予告なく運営チームにてコミュニティを削除・停止する場合があります。
- コミュニティ内で不適切な活動が続くメンバーがいる場合は、参加メンバー画面からそのユーザーをブロックすることができます。ブロックされたユーザーは、コミュニティから退出され、二度とコミュニティに参加することができなくなります。

ガイドラインおよびYOUTRUST利用規約に違反するようなコミュニティの使い方をしている場合、アカウント・コミュニティ削除対応をとることがあります。
ご自身の信頼を損なわないよう、メンバー・オーナーのことをお互いに尊重しあいながら、コミュニティを使って「つながり」を深めていただければ嬉しいです。
