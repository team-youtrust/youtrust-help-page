# カンパニーページについて知りたい

カンパニーページとは、**「会社の情報が蓄積されているページ」**です。

## カンパニーページで見れること

- 会社のビジョンや事業内容
- 会社の掲載募集
- 会社のメンバー
- ピックアップされたメンバーの投稿

## カンパニーページのフォローの仕方

PCwebからはホームの右側下部の「企業一覧を見る」から企業一覧が表示されます。気になる会社があれば、**「フォローする」ボタン**を押すだけで、フォローが完了します。

## カンパニーページをフォローするメリット

スカウトを受け取れる可能性が高くなる

企業のリクルーターはカンパニーページのフォロワーであれば、つながりの範囲に関係なくメッセージを送ることができます。フォローしておくことで、情報を逃さないだけでなく、気になる企業からのスカウトを**受け取りやすく**なります。
