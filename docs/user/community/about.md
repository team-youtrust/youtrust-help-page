# :material-information-outline: YOUTRUSTコミュニティについて知りたい

YOUTRUSTのコミュニティは、興味関心のある話題の中でユーザー同士で交流し、情報を共有する場所になります。

コミュニティメンバーは、コミュニティ内の会話を通じて交流を深めることができます。コミュニティに参加したメンバーだけが、コミュニティの中で会話をすることができます。

YOUTRUSTコミュニティでは、**「安心して、安全に楽しむことができるコミュニティ」**を目指しています。

そのために以下のルールを守り、コミュニティをお楽しみください。

- コミュニティメンバー同士、お互いに敬意を持って接しましょう。
- コミュニティを通じて発生するイベントにおいて、個人情報やお互いが交流を深めるために開示したパーソナルな情報はコミュニティの中や個々人の間で留めるようにしましょう。
- その他、YOUTRUST全体の利用規約を守った上で、ご参加・コミュニケーションください。

[利用規約](https://youtrust.jp/terms_of_service)
