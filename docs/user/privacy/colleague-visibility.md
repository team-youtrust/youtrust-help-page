# :material-account-eye: 同僚に副業・転職意欲が公開されているのか知りたい

プロフィールの「現在の所属」に同じ企業・組織名が記入されているユーザーには、
**あなたの副業意欲・転職意欲は公開されない**仕様になっています。

!!! warning "意欲非表示の注意点"
    1. **正式名称で入力がされていない場合、別の企業と見なされます**。意欲を表示させないために、必ず正式名称で入力してください。
    2. グループ会社や親/子会社など正式名称が異なる場合には別所属とみなされ、設定をしなければ意欲状況が公開されてしまいます。

「現在の所属」は異なるが、副業・転職意欲を非表示にしたい企業がある場合でも、設定をすれば意欲を非公開にすることができます。

詳細はこちらのページをご確認ください。

[意欲を特定の企業に非公開にしたい](hide-from-company.md)
