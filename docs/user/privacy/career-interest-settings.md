# :material-tune: 副業・転職意欲の設定/変更方法を知りたい

## :material-cellphone: アプリ

ホーム ＞ 右上の自分のアイコンをタップ ＞ 変更したい方の意欲の✐マークをタップ

![アプリ意欲変更](./images/app-career-interest-change.jpg){  width="500" style="display: block; margin: 0 auto;" }

## :material-monitor: Web

[マイページ](https://youtrust.jp/mypage?tab=profile) ＞ 変更したい方の意欲の✐マークをクリック

![Web意欲変更](./images/web-career-interest-change.png){ width="500" style="display: block; margin: 0 auto;" }

「現在の所属」が同じユーザーには副業・転職意欲は公開されないので、ご安心ください。
「現在の所属」が異なる場合でも、特定の企業に意欲を非公開にする設定ができます。

詳細はこちらをご覧ください。

[:material-eye-lock-outline: 意欲を特定の企業に非公開にしたい](hide-from-company.md)
