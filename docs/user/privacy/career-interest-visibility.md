# :material-eye-off-outline: 副業・転職意欲が見られない範囲を知りたい

マイページで設定している副業意欲・転職意欲は、以下の方々からは閲覧できません。

- **あなたの所属企業と同一のユーザー**
    - 例：「株式会社YOUTRUST」に所属している場合には、同じ「株式会社YOUTRUST」に所属しているユーザーからは意欲は見えません。
    - [:material-account-eye: 同僚に副業・転職意欲が公開されているのか知りたい](colleague-visibility.md)

- **非公開に設定している企業に所属しているユーザー**
    - 所属企業以外で意欲を非公開にしたい企業がある場合には設定が可能です。
    - [:material-eye-lock-outline: 意欲を特定の企業に非公開にしたい](hide-from-company.md)

- **あなたがブロックしているユーザー**
    - ブロックしたアカウントは、あなたのプロフィールを閲覧できなくなります。
    - [:material-block-helper: 「ブロックする」について知りたい](../connection/block.md)

- **あなたの友達でも公式リクルーター(採用担当者)でもないユーザー**
    - 友達としてのつながりもなく、公式リクルーターとして登録していないユーザーからは意欲を閲覧されることはありません。
    - [:material-badge-account-horizontal: 公式リクルーター(採用アカウント)とは何ですか？](../other/official-recruiter.md)
