# :material-shield-lock-outline: 情報セキュリティ方針

株式会社YOUTRUST（以下、当社）は、当社の情報資産、並びにお客様からお預かりした情報資産を事故・災害・犯罪などの脅威から守り、お客様ならびに社会の信頼に応えるべく、以下の方針に基づき全社で情報セキュリティに取り組みます。

## :material-numeric-1-circle: 経営者の責任

当社は、経営者主導で組織的かつ継続的に情報セキュリティの改善・向上に努めます。

## :material-numeric-2-circle: 社内体制の整備

当社は、情報セキュリティの維持及び改善のために組織を設置し、リスクアセスメント及びリスク対応を含む情報セキュリティ対策を社内の正式な規則として定めます。

## :material-numeric-3-circle: 従業員の取組み

当社の従業員は、情報セキュリティのために必要とされる知識、技術を習得し、情報セキュリティへの取り組みを確かなものにします。

## :material-numeric-4-circle: 法令及び契約上の要求事項の遵守

当社は、情報セキュリティに関わる法令、規制、規範、契約上の義務を遵守するとともに、お客様の期待に応えます。

## :material-numeric-5-circle: 違反及び事故への対応

当社は、情報セキュリティに関わる法令違反、契約違反及び事故が発生した場合には適切に対処し、再発防止に努めます。
