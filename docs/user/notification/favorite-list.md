# :material-heart-outline: 気になるリストについて知りたい

## :material-help-circle-outline: 気になるリストとは？

気になるリストとは、YOUTRUSTを有料でご契約いただいている法人企業が利用している**タレントプール機能です**。（タレントプールは、企業が中長期的に関わりを持ちたい方の情報を管理する仕組みを指しています。）

## :material-bell-ring-outline: 通知が届いたら

企業があなたを気になるリストに追加すると、通知が届きます。

リストに追加されたということは、その企業があなたに興味を持っているということです。

ぜひ、プロフィールや投稿などを充実させて、企業にあなたのことを知ってもらいましょう！
