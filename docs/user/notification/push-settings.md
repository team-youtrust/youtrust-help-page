# :material-cellphone-message: アプリのプッシュ通知設定を変更したい

## アプリ

ホーム＞アイコン＞メニューバー＞設定＞通知設定からアプリのプッシュ通知の設定を変更できます。

変更できる項目は、以下となっています。

### プッシュ通知設定の項目

- チームYOUTRUSTからのお知らせ
- 新着メッセージ
- つながり申請の受信
- つながり申請の承諾
- 知り合いかも
- 職歴と一緒に働いていた人への追加
- 紹介コメント
- 友達や候補者の副業意欲の更新
- 友達や候補者の転職意欲の更新
- 友達の初投稿
- 友達の新規投稿
- 友達の久しぶりの投稿
- 投稿へのいいね
- 投稿へのコメント
- 投稿に対するコメントへのいいね
- 投稿への「話を聞きたい！」
- 投稿メンション
- メッセージ返信のリマインド
- スカウトの再送をリマインド
- 足跡
- 新規フォロー
- 気になるリストに追加
- 募集へのいいね
- 承認制コミュニティへの参加申請
- 承認制コミュニティへの参加承認
- コミュニティ内の新着コメント・メンション
- コミュニティ内コメントへのいいね
