# 各種SNSアカウントと連携したい

## アプリ

ホーム＞自分のアイコンをタップ＞ 右上の✐マーク＞SNS連携

※ボタンをスライドするとWebページに遷移します。Webページよりログイン後、マイページ ＞ マイメニュー ＞ 設定画面 よりアカウント連携をお願いします。

## Web

[マイページ](https://youtrust.jp/mypage?tab=profile)や[設定](https://youtrust.jp/settings/linkage)から各種SNSのアカウント連携ができます。アカウントを連携することで、企業があなたのプロフィールに興味を持ちやすくなります。
連携するとプロフィールにアイコンが表示されます。

一度連携した後は、マイページ＞SNS連携から追加できます。

### 連携できるSNSアカウント（2025年8月時点）

- Facebook
- LINE
- Google
- X（Twitter）
- LinkedIn
- GitHub
- Zenn
- Qiita
- note
- Wantedly

※Facebook連携のみ一度連携すると解除ができないためご注意ください。新しいアカウントにFacebook連携したい場合、紐づいている古いアカウントを削除していただく必要がございます。

削除については[アカウントを退会したい](../account/delete.md)のページをご覧ください。
