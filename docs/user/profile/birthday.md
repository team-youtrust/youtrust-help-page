# 生年月日について知りたい

## :material-cake-variant: 生年月日の設定について

2023/12よりYOUTRUSTで生年月日を登録できるようになりました。  
登録した生年月日は公開と非公開を選択できるようになります。

公開設定は「つながりに公開」と「非公開」があります。  
公開する場合はプロフィールページに表示されます。「つながりに公開」を選択している場合でも「年」は表示されずに「月日」のみが表示されるようになります。

![プロフィールカードサンプル](./images/profile-card-sample.png){ width="400" style="display: block; margin: 0 auto;" }

## :material-eye-settings: 生年月日の公開設定と公開範囲

|                          | 非公開 | つながり（友達）まで公開 |
| ------------------------ | ------ | ------------------------ |
| 自分                     | ✗      | ◯                        |
| 直接のつながり（＝友達） | ✗      | ◯                        |
| つながっていない         | ✗      | ✗                        |

一度登録した生年月日を手元で修正することはできません。  
もし誤って入力してしまった場合には、お問い合わせ窓口までご連絡くださいませ。
YOUTRUST運営にて修正いたします。
