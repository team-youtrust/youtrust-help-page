# :material-comment-account-outline: 紹介コメントを書きたい

紹介コメントとは、友達や同僚をフリーテキストで紹介できる機能です。

印象に残っているエピソードや、他の人が知らない逸話などを書いて、紹介しましょう！

![紹介コメント例](./images/web-recommendation-comment-example.png){ width="400" style="display: block; margin: 0 auto;" }

## :material-pencil-outline: 紹介コメントの書き方

1. 紹介コメントを書きたい相手のマイページを見る
2. 紹介コメントのページを見る
3. 「紹介コメントを書く」をタップ

![紹介コメントの書き方](./images/web-recommendation-comment-guide.png)
