# :material-account-remove: アカウントを退会したい

YOUTRUSTのアカウントは、アプリとWebから退会手続きが可能です。

## :material-monitor: Web
「編集 > アカウント管理」から、退会処理を行うことができます。

## :material-cellphone: App
「歯車マーク > アカウント管理 > 退会処理」から、退会処理を行うことができます。

## :material-alert-circle-outline: 退会時の注意事項

- 一度退会したアカウントを復活させることはできません。
- **公式リクルーターの権限が付与されているアカウント**は、上記の手順では退会できません。お手数ですが[こちら](https://lp.youtrust.jp/service_contact)からお問合せいただくか、担当のカスタマーエクスペリエンス（CX）担当者までお問い合わせください。
