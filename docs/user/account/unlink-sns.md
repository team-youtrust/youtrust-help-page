# :material-link-off: SNSアカウント連携を解除したい

連携しているSNSアカウント（Facebook、Google、LINE）の連携を解除する方法をご案内します。

## アプリ

ホーム画面のご自身の**アイコン**をタップし、マイページに移動してください。次に、画面右上の**設定（歯車アイコン）をタップし、設定画面から外部連携>SNS連携**へ進むと、連携の解除が行えます。

## Web

こちらの[設定画面](https://youtrust.jp/settings/account)の**外部連携**から、連携の解除が行えます。

---

## 解除ができない場合

**2つ以上のログイン方法が設定されている場合のみ**、SNSアカウントの連携を解除することができます。

例えば、以下のような組み合わせが必要です。

- メールアドレスとパスワード + SNSアカウント（Facebook、Google、LINEなど）が連携済み
- 2つ以上のSNSアカウントが連携済み
