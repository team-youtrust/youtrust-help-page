# :material-email-edit: 登録アドレスを変更したい

## アプリ

マイページ→右上のバー→設定→アカウント管理→メールアドレスから
メールアドレスの変更が可能です。

確認メールを送信し認証が完了すると、新しいメールアドレスが有効になります。

## Web

こちらの[設定画面](https://youtrust.jp/settings)から、登録メールアドレスを変更できます。
確認メールを送信し認証が完了すると、新しいメールアドレスが有効になります。
