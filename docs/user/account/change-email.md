# :material-email-edit: 登録アドレスを変更したい

YOUTRUSTに登録しているメールアドレスは、アプリとWebの両方から変更できます。

## :material-cellphone: アプリ

ホーム画面右上のご自身の**アイコンをタップ**し、マイページに移動します。 次に、画面右上の**設定（歯車アイコン）をタップし、設定画面からアカウント管理**へ進んでください。 メールアドレスの項目をタップすると変更画面に移動しますので、新しいメールアドレスを入力してください。 新しいメールアドレスに送信される認証メールを確認し、認証を完了させると、変更が有効になります。

## :material-monitor: Web

こちらの[設定画面](https://youtrust.jp/settings/account)から、登録メールアドレスを変更できます。
確認メールを送信し認証が完了すると、新しいメールアドレスが有効になります。
