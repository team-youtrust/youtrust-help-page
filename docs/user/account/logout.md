# :material-logout: アカウントからログアウトしたい

ログアウトは、アプリとWebの両方から行うことができます。

## :material-cellphone: アプリ / Web

「設定 > アカウント管理」から、ログアウトできます。

![アプリのアカウント管理画面とログアウトボタン](images/app-account-management-logout.png){ width="50%" style="display: block; margin: 0 auto;" }

## :material-alert-circle-outline: 注意事項

YOUTRUSTの退会をご希望の場合は、[:material-account-remove: アカウントを退会したい](delete.md)のページをご覧ください。
