# :material-login-variant: ログインできない時の解決方法を知りたい

ログインできない原因はいくつか考えられます。以下の4つのパターンをご確認ください。

## :material-numeric-1-circle: 1. アカウント新規登録画面に入っている

ログインページとアカウント新規登録ページを間違えている可能性があります。

画面に「ログイン」と表示されているか確認し、登録ページではなく[ログインページ](https://youtrust.jp/sign_in?type=sign_in)からログインを試してください。

![YOUTRUSTログインページ](images/web-login-page.png)

## :material-numeric-2-circle: 2. ログイン方法を間違えている

アカウント作成時に利用した方法と異なる方法でログインしようとしていないか確認してください。

たとえば、Gmailアドレスで登録しても、Googleアカウント連携を設定していなければ「Googleでログイン」は使用できません。

ログインには、メールアドレスとパスワードを使用してください。

## :material-numeric-3-circle: 3. パスワード再設定メールが届かない

SNS連携（Facebook、Google、LINE、Apple ID）のみでアカウントを作成している場合、メールアドレスでのログイン設定はされていません。このため、パスワード再設定のメールは届きません。

メールアドレスでログイン設定をしていない方は、各SNSログインボタンからログインをお試しください。

## :material-numeric-4-circle: 4. 新しいアカウントにログインできない

以前のログイン情報がブラウザに残っていると、新しいアカウントでログインできない場合があります。 新しくアカウントを作り直した場合は、以下の手順でキャッシュとCookieを消去してください。

### :material-cellphone: アプリ

アプリを一度アンインストールし、再度インストールしてログインを試してください。

### :material-monitor: Web

[こちら](https://support.google.com/accounts/answer/32050?hl=ja&co=GENIE.Platform%3DDesktop)でCookieの消去方法について詳しく確認できます。
