# :material-hand-wave: 「話を聞きたい！」の流れを知りたい

「話を聞きたい！」ボタンを押すと、募集を出している企業・個人へあなたがその募集に興味があることが通知されます。

募集先に自分のことを興味を持ってもらい理解してもらいやすくするために、
可能な限りプロフィール情報を充実させて「話を聞きたい！」ボタンを押すようにしましょう。

[プロフィールの追加や編集をしたい](../profile/edit.md)

募集先があなたのことに興味を持ってくれるとメッセージが届きます。
（右下📧アイコンに通知マークが表示されます）

メッセージをやり取りし、カジュアル面談などのお互いを理解しあう場を設けてもらいましょう！

もちろん話を聞いてみて、ご自身の条件や希望と違うと思ったら遠慮なくお断りをしていただいて構いません。面談や契約の進め方で不安があれば、お気軽にYOUTRUSTお問い合わせ窓口までご連絡ください。

![アプリの話を聞きたいからメッセージまでの流れ](images/app-want-to-talk-flow.png)

また「話を聞きたい！」ボタンを押したからといって、
残念ながら必ずしも企業・個人から返信が来るとは限りません。

メッセージが来ない場合は、まずプロフィールの充実を進めてみてください。
プロフィールが白紙のため連絡を取るべきか迷っているかもしれません。
