# :material-domain: カンパニーページについて知りたい

カンパニーページとは、**「会社の情報が蓄積されているページ」**です。

## :material-eye-outline: カンパニーページで見れること

- 会社のビジョンや事業内容
- 会社の掲載募集
- 会社のメンバー
- ピックアップされたメンバーの投稿

![Webのカンパニーページ概要](images/web-company-page-overview.png)

## :material-heart-plus-outline: カンパニーページのフォローの仕方

![カンパニーページのフォロー導線](images/web-company-page-follow-guide.png)

PCwebからはホームの右側下部の「企業一覧を見る」から企業一覧が表示されます。気になる会社があれば、**「フォローする」ボタン**を押すだけで、フォローが完了します。

## :material-star-outline: カンパニーページをフォローするメリット

スカウトを受け取れる可能性が高くなる

企業のリクルーターはカンパニーページのフォロワーであれば、つながりの範囲に関係なくメッセージを送ることができます。フォローしておくことで、情報を逃さないだけでなく、気になる企業からのスカウトを**受け取りやすく**なります。
