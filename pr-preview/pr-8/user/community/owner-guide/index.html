    コミュニティオーナーが出来ることを知りたい¶

    コミュニティの作成者である「コミュニティオーナー」は、各コミュニティのコミュニケーションが活発になり、安心・安全に楽しむことができるようにコミュニティの運用をお願いします。

    コミュニティオーナーはコミュニティ運用のために以下の機能が利用できます。

    参加申請の承認/削除¶

    承認制コミュニティを運用している場合は、コミュニティ参加希望者から申請が届きます。申請一覧画面より、参加申請の承認/削除ができます。

    コミュニティ情報の編集¶

    コミュニティのカバー画像・タイトル・説明文を編集可能です。コミュニティが魅力的に伝わるように分かりやすいタイトル/説明文の記載をおすすめしています。

    ※コミュニティ情報の編集において、当社または第三者の知的財産権、肖像権、プライバシー権、名誉その他の権利を侵害するまたはその恐れのある内容に当たらないようご注意ください。

    公開設定の変更¶

    コミュニティの公開設定を「全体公開」「承認制」のいずれかに変更できます。

    承認制から全体公開に変更する際に、参加申請が届いている場合は、自動的にすべて承認されます。

    ※公開設定の変更はWebのPC版からのみ操作可能

    トピック/コメントの削除¶

    コミュニティ内で不適切なトピックやコメントがある場合は、そのトピック/コメントをオーナー権限で削除することができます。削除されたユーザーに通知は届きません。

    参加メンバーのブロック¶

    コミュニティ内で不適切な行動が続く方は、コミュニティからブロックすることが可能です。

    コミュニティからブロックしたユーザーは、今後同じコミュニティに参加することができなくなります。（ブロックされたユーザーのコミュニティ一覧や検索にもヒットしなくなります。）

    共同オーナーの指定¶

    コミュニティの参加メンバーの中から、一緒にコミュニティを運営していく共同オーナーを指定することができます。コミュニティ作成者の初期オーナーと、後から追加された共同オーナーは同じ権限を持ちます。

    ※コミュニティのオーナーは最大10人となります

    ※1度オーナーに指定した後にメンバーに戻すことも可能です（最低1人のオーナーが必要）

    今後もオーナーの方がコミュニティを運用しやすくするために随時機能を追加してまいります。

    希望の機能がある際には、YOUTRUSTコミュニティオーナーの会にて投稿をお願いします。
