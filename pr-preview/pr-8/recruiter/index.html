    公式リクルーター向けヘルプページ¶

    こちらは、YOUTRUSTを有料でご利用いただいている企業様向けのヘルプページです。

    基本の活用マニュアル¶

    • 投稿する
    • 募集のステータス管理をする
    • 候補者を増やす
    • 候補者を検索する
    • 候補者を管理する
    • スカウトを送る
    • 通知設定を管理する

    お問い合わせ窓口¶

    ご不明点はこちらのお問い合わせフォームよりご連絡ください。 3営業日を目処にお返事いたします。

    ▼問い合わせフォーム https://lp.youtrust.jp/service_contact

    FAQ¶

    投稿¶

    • 投稿の閲覧範囲を知りたい
    • 「話を聞きたい」を押してくれた人を確認したい
    • 投稿をカンパニーページに紐づけたい
    • 利用規約や法令に違反となる事例を知りたい
    • 雇用契約を伴う求人掲載時の注意点を知りたい
    • 投稿画像の推奨サイズを知りたい
    • 公開した募集を編集したい

    カンパニーページ¶

    • カンパニーページを編集したい
    • カンパニーページメンバーを追加したい
    • カンパニーページのフォロワーについて知りたい
    • カンパニーページのフォロワーを閲覧したい
    • カンパニーページのフォロワーを増やしたい
    • カンパニーページのフォロワーへ会社名義で通常投稿したい

    友達¶

    • つながりを増やしたい
    • つながり候補に出てくる人の条件を知りたい
    • つながり申請の温度感を知りたい

    リクルーター管理画面¶

    • 公式リクルーター権限の追加方法を知りたい
    • リクルーター権限の各役割の違いについて知りたい
    • 誰が管理画面の閲覧権限があるのか知りたい
    • 足跡通知について知りたい
    • 管理画面内の通知について知りたい
    • 登録直後のホットなタレントβ版とはなにか知りたい
    • リクルーター管理画面を他の所属に切り替えたい
    • リクルーター管理画面をスマホから使いたい

    候補者の検索¶

    • 候補者検索で出てくる範囲を知りたい
    • 候補者探しのコツを知りたい
    • 検索条件を保存したい
    • そっくり検索とは何か知りたい
    • AIタレントリクエストについて知りたい
    • 「つながりの近いリクルーター」とは何か知りたい
    • 「新着ユーザー順」とは何か知りたい
    • イチオシタレントとは何か知りたい
    • 「○人が興味あり」とは何か知りたい
    • アプローチできる候補者を増やしたい
    • 検索画面から特定のユーザーを非表示にしたい
    • 検索画面に他のリクルーターのつながりが反映されない

    候補者の管理¶

    • 気になるリストについて知りたい
    • 気になるリストのアクセスを管理したい
    • リストに追加すると相手に通知されるのか知りたい
    • 「気になるリスト」と「候補者リスト」の違いを知りたい
    • 候補者のリストから候補者を外したい
    • 候補者のリストの進捗状況が「スカウト済み」にならない

    通知¶

    • サービス内の通知について知りたい
    • 意欲変更通知について知りたい
    • Slackに通知を連携したい
    • 採用管理ツールと連携したい

    スカウト・チャット¶

    • スカウトデータをcsvダウンロードしたい
    • メッセージを送れる範囲について知りたい
    • スカウトとメッセージの違いを知りたい
    • スカウトが送れないと表示される
    • 他リクルーターのスカウト内容を確認したい
    • さくさくメッセージビルダー(α版)について知りたい
    • 予約送信機能について知りたい

    候補者とのやりとり¶

    • グループチャット/グループスカウトを作りたい
    • カジュアル面談とは何か知りたい
    • いつ労働条件を示す必要があるのか知りたい
    • リクルーターが退職後の候補者管理について知りたい
    • 過去送付したスカウトがユーザー画面側で非表示になるケースを知りたい
    • 公式リクルーター権限を外した後の実際のメッセージ画面について知りたい

    請求・契約¶

    • 企業プラン導入にあたっての事前準備
    • 契約内容を確認したい
    • 送付可能なスカウト通数を確認したい
    • 契約プランを途中で変えたい
    • 請求書がいつ届くのか知りたい
    • クレジットカードで支払いたい
    • 追加購入したスカウトを契約更新後に持ち越したい

    タレントピック¶

    • タレントピックとは何か知りたい
    • タレントピックを3人以上閲覧したい
